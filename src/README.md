# Jump Jump Revolution

A 2D arcade platformer game built in Java featuring multiple themed levels, enemies, and powerups.

## Game Features
- Three unique themed levels: Desert, Christmas, and Halloween
- Dynamic enemy AI with shooting mechanics
- Power-up system with different abilities
  - Speed boost
  - Double jump
  - Health increase
  - Invincibility
- Themed graphics for each level
- Background music and sound effects
- Score system

## Controls
- Left Arrow: Move left
- Right Arrow: Move right
- Up Arrow: Jump
- ESC: Pause menu

## Setup & Running the Game

### Prerequisites
- Java Development Kit (JDK) installed

### Running the Game
1. Open command prompt in the project directory

2. Compile the source files:
```bash
javac -d target/classes src/main/java/*.java

3. Run the game
